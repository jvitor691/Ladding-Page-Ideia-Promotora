
# Landing Page - Ideia Promotora

Esta landing page foi desenvolvida para a **Ideia Promotora**, uma empresa correspondente bancária autorizada pelo **Banco Central do Brasil**. O objetivo desse projeto foi criar uma **página institucional** que transmita **credibilidade**, **segurança jurídica** e **transparência** aos clientes da empresa, promovendo os serviços de crédito consignado, cartões e saque do FGTS.

## Descrição do Projeto

A **Ideia Promotora** é especializada na intermediação de **operações financeiras**. A landing page foi criada para apresentar os **produtos financeiros** oferecidos, os **bancos parceiros** e as **informações de contato** da empresa. O design foi cuidadosamente planejado para proporcionar uma navegação **simples** e **intuitiva**, com foco na experiência do usuário.

### Objetivos:
- **Credibilidade**: Apresentar informações claras e objetivas sobre os serviços oferecidos pela Ideia Promotora.
- **Segurança jurídica**: Garantir que os visitantes compreendam que a empresa é regulamentada pelo Banco Central.
- **Transparência**: Explicar de forma simples os processos de intermediação de crédito e outros serviços.
- **Agilidade**: Garantir uma navegação rápida e eficiente, com formulário de contato e acesso aos produtos financeiros.

## Funcionalidades

- **Produtos financeiros**: Destaque para os produtos como crédito consignado e cartões.
- **Bancos parceiros**: Lista de instituições financeiras parceiras.
- **Formulários de contato**: Formulário para que os clientes possam iniciar o processo de solicitação de crédito ou parcerias.
- **Design responsivo**: Garantindo uma boa experiência em dispositivos móveis e desktops.

## Tecnologias Utilizadas

- **React.js**: Biblioteca JavaScript para construir interfaces de usuário dinâmicas.
- **Vite**: Ferramenta de build rápida e eficiente, usada para otimizar o processo de desenvolvimento.
- **CSS customizado**: Estilos específicos, com foco em acessibilidade e responsividade.
- **React Router**: Para navegação entre as seções da landing page.

## Como Rodar o Projeto Localmente

1. **Clone o repositório**:

   ```bash
   git clone https://github.com/jvitor691/Ladding-Page-BRIEFING.git
   cd Ladding-Page-BRIEFING
   ```

2. **Instale as dependências**:

   ```bash
   npm install
   ```

3. **Execute o servidor de desenvolvimento**:

   ```bash
   npm run dev
   ```

4. Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Processo de Desenvolvimento

- **Planejamento**: O primeiro passo foi entender os requisitos da empresa e quais serviços seriam destacados na landing page.
- **Design**: Criação do layout da página, incluindo escolha de paleta de cores, tipografia e a disposição dos elementos.
- **Desenvolvimento**: Implementação da página utilizando **React.js** e **Vite**, com foco em uma boa experiência de navegação e performance.
- **Responsividade**: A página foi projetada para ser totalmente **responsiva**, com ajustes de layout para dispositivos móveis e tablets.
- **Testes**: Realização de testes de performance e compatibilidade entre navegadores.
